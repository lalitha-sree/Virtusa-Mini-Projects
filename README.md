# Virtusa-Mini-Projects
This repository contains multiple mini projects built to practice basic programming concepts across Python, Java, and SQL.Each project is simple, but covers important fundamentals like input handling, validation, and structured logic.

# Projects Included
**Password Validator (Java)**

This program validates user-entered passwords based on a few basic rules such as minimum length, inclusion of uppercase letters, and digits. If the password does not meet the criteria, it provides feedback and asks the user to try again.

**CityCab Fare Calculator (Python)**

A command-line based application that calculates cab fares based on distance, selected vehicle type, and time of travel. It also includes surge pricing during peak hours and displays the final result in a receipt-style format.

**Digital Library Database (SQL)**

A simple database design for a digital library system. It includes table creation and relationships to organize and manage data effectively.
